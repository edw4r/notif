# Notif demo website

This repository hosts the Notif demo website and privacy policy with Astro, React, and Tailwind CSS.

## Local development

Install dependencies once:

```sh
npm install
```

Start the local dev server:

```sh
npm run dev
```

Build the static website:

```sh
npm run build
```

Preview the built website:

```sh
npm run preview
```

## Project structure

- `src/pages/index.astro` is the product landing page.
- `src/pages/privacy-policy.astro` is the privacy policy page.
- `src/components/` contains reusable React components.
- `src/styles/global.css` contains Tailwind and site-wide CSS.
- `docs/` is generated by `npm run build` and is the GitHub Pages output folder.
